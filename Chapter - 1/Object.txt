----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Object :-  

    Object is a Non-Primitive DataTypes in Javascript.

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

    1) 'var' Object :-  
            You can change dataType of 'var' Object.
            We can Add Data in 'var' Object.
            We can Modify Data present in 'var' Object.

    2) 'let' Object :-  
            You can change dataType of 'let' Object.
            We can Add Data in 'let' Object.
            We can Modify Data present in 'let' Object.

    3) 'const' Object :-  
            You cann't change dataType of 'const' Object.
            We can Add Data in 'const' Object.
            We can Modify Data present in 'const' Object.
