What is Javascript :- 
    1) JavaScript is the programming language of the web.
    2) It can update and change both HTML and CSS.
    3) It can calculate, manipulate and validate data.

----------------------------------------------------------------------------------------------------------------------------------------

Note :- 
    1) JavaScript accepts both double and single quotes.
        - document.getElementById("demo").innerHTML = "Hello JavaScript";
        - document.getElementById('demo').innerHTML = 'Hello JavaScript';
    2) JavaScript and Java are completely different languages, both in concept and design.
    3) JavaScript was invented by Brendan Eich in 1995, and became an ECMA standard in 1997.
    4) ECMA-262 is the official name of the standard. ECMAScript is the official name of the language.
