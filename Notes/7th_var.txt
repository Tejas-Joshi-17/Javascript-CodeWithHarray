Difference Between var, let and const

  +---------+-------------------------+-------------+-----------+------------+----------------+
  |         |  Scope	                |  Redeclare	|  Reassign	|   Hoisted	 |   Binds this   |
  +---------+-------------------------+-------------+-----------+------------+----------------+
  |  var	  | global/function scope	  |   Yes	      |  Yes	    |   Yes	     |     Yes        |
  |  let	  |  blocked scope	        |    No       |  Yes	    |   No	     |      No        |
  |  const	|  blocked scope	        |    No       |  No	      |  No	       |      No        |
  +---------+-------------------------+-------------+-----------+------------+----------------+
